🛡️ CrowdPay: Democratic Communal Savings
Democratic communal savings with unanimous consensus protection.

CrowdPay is a high-impact digital product designed to revolutionize group savings through transparency and identity-first security. Built for the IDC x Enyata Hackathon, it leverages a robust frontend architecture and serverless backend to ensure group funds are protected by the collective.

## 🛠️ The Technical Stack
- **Frontend**: Vite + React + TypeScript (Strict Mode)
- **Styling**: Vanilla CSS + Tailwind (Premium Glassmorphism Design)
- **Database/Auth**: Firebase Firestore & Firebase Auth
- **Identity**: Interswitch API Marketplace (KYC & NIN Verification)
- **State Management**: React Context API & Real-time Firestore Listeners
- **Animations**: GSAP & Framer Motion for premium UX

## 💎 Key Features
- **Social Dashboard**: Real-time visibility into group contributions and goals.
- **Identity Verified (KYC)**: 1:1 NIN verification via Interswitch Marketplace.
- **Democratic Governance**: Unanimous consensus logic for withdrawals ("No Cap" Security).
- **Merchant Hub**: Automated payouts to verified vendors for product-linked jars.
- **Transaction Ledger**: Immutable tracking of every Naira in the ecosystem.

## 🧪 Installation & Local Setup
1. **Clone the Repo**:
   ```bash
   git clone https://github.com/Fatokz/Talent-board.git
   ```
2. **Navigate to Client**:
   ```bash
   cd crowdpay/client
   ```
3. **Install Dependencies**:
   ```bash
   npm install
   ```
4. **Run Development Server**:
   ```bash
   npm run dev
   ```
